Interfaz de Línea de Comandos

Aquí encontrará los comandos y ejemplos escenciales para poder utilizar el framework e instalar las aplicaciones que necesite.

Comandos Disponibles

Gestión de Repositorios

Puedes clonar un repositorio que contenga aplicaciones compatibles, y gestionarlos con comandos "git" para mayor facilidad.

git clone <url>

Las aplicaciones compatibles deben tener una nomenclatura específica, deben iniciar con un nombre de paquete, seguido del texto "Apps".

Por ejemplo, el repositorio de aplicaciones del mismo framework Irbis es:

git clone https://github.com/dulivu/IrbisApps.git
Navegación y Listado

Puedes listar las aplicaciones disponibles con el siguiente comando

ls apps

Las aplicaciones ya instaladas se marcarán en verde.

"ls" se comporta como el comando tradicional de Unix/Linux. Por lo que puedes enviar opciones adicionales para filtrar o modificar la salida.

Por ejemplo, para listar las aplicaciones dentro de un paquete específico:

ls -d <paquete>/*/
Instalación y Gestión

Ahora puedes instalar tu primera aplicación con el comando "install" seguido del nombre del paquete y la aplicación en el siguiente formato <paquete>/<aplicación>, ejemplo:

install IrbisApps/Cms

De igual manera, puedes quitar una aplicación instalada con el comando "remove". Ejemplo:

remove IrbisApps/Cms

⚠️ Elimina el terminal. Úsalo solo cuando hayas terminado de configurar tus aplicaciones.

Esto impedirá que puedas acceder a él nuevamente. Cuando hayas terminado de configurar tus aplicaciones. Tienes que correr este comando por seguridad.

remove terminal
Otros comandos

Para revisar la versión del framework

version

Para limpiar la pantalla del terminal

clear

Para ver la fecha y hora del servidor

date

Para ver el usuario actual del sistema

whoami

Para ver la información detallada de un módulo, ejemplo:

help IrbisApps/Cms
